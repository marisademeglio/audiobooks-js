Schema tester tests

These are manifest files that I had lying around for testing a library. They're not all valid and many have errors that would not be picked up by a schema validator, but rather a processor. That is to say, this is not a comprehensive set of files for testing schema. It is, however, a whole bunch of manifest files. Click 'run' to check a file against the schema.
